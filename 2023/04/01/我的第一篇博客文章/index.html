Hexo博客搭建并部署至Github

发表于2023-04-01|更新于2023-04-01
|阅读量:

第一步 安装相关内容

node -v #查看node版本
npm -v #查看npm版本
npm install -g cnpm –registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器
cnpm -v #查看cnpm版本
cnpm install -g hexo-cli #安装hexo框架
hexo -v #查看hexo版本

第二步 创建本地hexo博客

mkdir blog #创建blog目录
cd blog #进入blog目录
sudo hexo init #生成博客 初始化博客
hexo s #启动本地博客服务
localhost:4000 #本地访问地址

hexo n “我的第一篇文章” #创建新的文章

hexo clean #清理
hexo g #生成

第三步 将博客部署到github上

#github创建一个新的仓库 仓库名字必须为username.github.io

cnpm install –save hexo-deployer-git #在blog目录下安装git部署插件

#配置_config.yml
#冒号: 后面都要加空格
#windows下type后面的git要用单引号包住
#branch后面加main而不是master(2023/4/1)

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
      type: 'git'
     repo: https://github.com/username/username.github.io.git
      branch: main

#设置github的邮箱和用户名（部署出错记得加这两句）
git config –global user.email “xxx@email”
git config –global user.name “username”

hexo d #部署到Github仓库里
https://username.github.io/ #访问这个地址可以查看博客

更新博客后的三步走

hexo clean #清理
hexo g #生成
hexo d #部署到Github仓库里

更换博客主题

#下载yilia主题到本地 或者 手动下载并解压至themes下，并将文件夹名改为yilia
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia #下载yilia主题到本地

#修改hexo根目录下的 _config.yml 文件 ： theme: yilia

#然后三步走
hexo clean #清理
hexo g #生成
hexo d #部署到Github仓库里
